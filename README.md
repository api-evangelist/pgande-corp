# PG&E Corporation (pgande-corp)
PG&E Corporation is the parent company of Pacific Gas and Electric Company, an electric and natural gas utility serving customers in northern and central California. This repository captures the APIs, developer tools, and machine-readable API artifacts for PG&E Corporation. Operational APIs and developer resources for the utility are surfaced under the Pacific Gas and Electric Company (pg-and-e) listing.
